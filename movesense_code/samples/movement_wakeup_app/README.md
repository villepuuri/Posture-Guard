# Movement Wakeup sample (movement_wakeup_app)

This sample shows how the sensor can be set for movement wakeup and how to put the sensor to "FullPowerOff" mode.

## What it does:
 * Starts a timer that blinks LED
 * subscribes to whiteboard route table
 * If no WB connection is ongoing, sets up movement wakeup and puts the sensor to FullPowerOff mode
